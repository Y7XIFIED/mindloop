## Overview
Mindloop is a motion-first SaaS landing page built with React, TypeScript, and Vite.
It blends immersive video sections, smooth interactions, and responsive layout behavior into one focused one-page experience.

Live site: [https://mindloop-silk.vercel.app](https://mindloop-silk.vercel.app)

## Features
- Motion-led hero and sectional storytelling
- Smooth in-page navigation and responsive interaction patterns
- Real-time styled call-to-action sections with video backgrounds
- Componentized React architecture with reusable UI primitives
- Tailwind-powered design system with fast iteration workflow
- Ready for static deployment on Vercel

## Installation
1. Clone the repository.

```bash
git clone https://github.com/Y7XIFIED/mindloop.git
```

2. Move into the project directory.

```bash
cd mindloop
```

3. Install dependencies.

```bash
npm install
```

4. Start the development server.

```bash
npm run dev
```

## Usage
- Run `npm run dev` for local development.
- Run `npm run build` to create a production build.
- Run `npm run preview` to preview the production output locally.
